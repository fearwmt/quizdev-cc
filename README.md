# Student & Classroom Management System

ระบบจัดการนักเรียนและห้องเรียน (NestJS + React + Ant Design + MySQL)

## 🔧 Requirements
- Node.js v18+
- MySQL
- NPM

## 🚀 การรัน Project

```bash
# Backend
cd backend
npm install        
npm run start:dev
```

```bash
# Frontend
cd frontend
npm install
npm start
```
